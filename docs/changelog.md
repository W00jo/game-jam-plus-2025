# Changelog

## [Initial Setup] - 2025-01-14

### Added
- Initial Godot 4.5 project setup
- Split-screen co-op functionality with 2 viewports
- Ground player character with WASD + Sprint controls
- Sniper camera with top-down view and mouse aiming
- Basic 3D environment (floor, walls, lighting)
- Objective and item collection system
- Global GameManager for score tracking
- Player input mappings for both players
- Visual crosshair for sniper scope
- Zoom controls for sniper camera

### Game Features
- Player 1: Third-person ground movement with physics
- Player 2: Top-down sniper view following Player 1
- Interactive objectives and collectible items
- Score system with points for objectives and items
- Collision layers for world, players, objectives, and enemies
